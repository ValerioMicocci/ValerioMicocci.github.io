COMANDI MS-DOS

  1. cd:Visualizza il nome della directory corrente o modifica
  2. cls:Cancella la finestra del prompt dei comandi.
  3. cmd:Avvia una nuova istanza dell'interprete dei comandi
  4. copy:Copia di uno o più file da una posizione a un'altra.
  5. date:Visualizza o imposta la data di sistema
  6. del:Elimina uno o più file.
  7. deltree:Elimina la directorye tutto ciò che contiene.
  8. dirVisualizza un elenco di file e le sottodirectory della directory
  9. echo:Visualizza i messaggi o attiva o disattiva la ripetizione dei comandi
  10. exit:Chiude l'interprete del comando o lo script batch corrente
  11. md:Crea una directory o una sottodirectory
  12. move:Sposta uno o più file da una directory a un'altra directory.
  13. rd:Sposta uno o più file da una directory a un'altra directory.
  14. renRinomina file o directory.
  15. time:Visualizza o imposta l'ora di sistema
  16. ver:Visualizza la versione del sistema operativo
